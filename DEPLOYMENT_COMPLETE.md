**Исправлено:** В скрипте деплоя база данных была исключена из копирования (`--exclude='bookme.db'`), поэтому при деплое использовалась старая БД с сервера.

**Решение:** Актуальная локальная БД была скопирована на сервер вручную, заменив старую версию. Теперь все пользователи и данные из локальной БД доступны на сервере.

**Рекомендация:** При следующем деплое, если нужно обновить БД, явно скопируйте её командой:
