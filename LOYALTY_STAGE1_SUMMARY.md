### 3. ✅ Исправлена авторизация

**Изменения:**
- Все функции загрузки данных проверяют `token && isAuthenticated` перед запросами
- Обработчики создания/обновления/удаления показывают Alert, если нет токена
- Нет запросов к API без токена (защита от 401)

**Проверки:**
```typescript
// В loadDiscounts()
if (!hasLoyaltyAccess || !token || !isAuthenticated) {
  setDiscountsLoading(false);
  return;
}

// В обработчиках
if (!token || !isAuthenticated) {
  Alert.alert('Ошибка', 'Необходима авторизация');
  return;
}
```
