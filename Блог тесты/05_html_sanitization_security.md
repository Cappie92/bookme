# Проверка XSS и опасного HTML

Ниже есть несколько вставок, которые система должна обработать безопасно.

<script>alert('xss')</script>

<div onclick="alert('hack')">Кликни меня</div>

<a href="javascript:alert('bad-link')">Опасная ссылка</a>

Безопасная ссылка для сравнения: [DeDato](https://dedato.ru)

## Обычный HTML

<b>Жирный текст</b> и <i>курсив</i> могут либо сохраниться, либо быть преобразованы безопасно — это ок.

## Код

```html
<script>alert("inside code block")</script>
```

Главное, чтобы опасные скрипты и события не оказывались в итоговом HTML как исполняемый код.
