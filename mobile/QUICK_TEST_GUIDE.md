## 🚀 Запуск (3 команды)

### Вариант 1: Автоматический скрипт (рекомендуется)

```bash
cd mobile
./start-ios-test.sh
```

Затем в терминале нажмите **`i`** когда появится QR-код.

### Вариант 2: Вручную

#### Шаг 1: Запустить симулятор
```bash
open -a Simulator
```

#### Шаг 2: Запустить Expo
```bash
cd mobile
npm start
```

#### Шаг 3: Открыть в симуляторе
В терминале где запущен `npm start` нажмите:
```
i
```

## 🔍 Отладка

### Просмотр логов

**Metro Bundler (терминал):**
- ✅ `🌐 [API] Запрос: GET /api/...` - запрос отправлен
- ✅ `🔑 [API] Токен добавлен` - авторизация работает
- ❌ `❌ [API] Ошибка: ...` - ошибка запроса

**iOS Simulator:**
- Нажмите `Cmd + D` → "Debug Remote JS" для открытия DevTools
- Нажмите `Cmd + R` для перезагрузки

### Частые проблемы

**"Network Error":**
- Проверьте backend: `curl http://localhost:8000/docs`
- Проверьте `.env`: `cat mobile/.env`

**Приложение не открывается:**
- Проверьте симулятор: `xcrun simctl list devices | grep Booted`
- Перезапустите Expo: `npm start -- --clear`
