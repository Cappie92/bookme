# Подключение Expo Go на iOS

## Проблема: нет кнопки сканирования QR-кода

В новых версиях Expo Go на iOS кнопка сканирования может быть в другом месте или нужно использовать другой способ подключения.

## Решение 1: Поиск кнопки сканирования

1. **Откройте Expo Go** на iPhone
2. **Проверьте главный экран** - должна быть кнопка "Scan QR code" или иконка камеры
3. **Проверьте вкладку "Projects"** - там может быть кнопка "+" или "Scan QR"
4. **Проверьте вкладку "Profile"** - там может быть опция сканирования

## Решение 2: Использование URL напрямую

1. **Запустите Expo:**
   ```bash
   cd /Users/s.devyatov/DeDato/mobile
   npx expo start
   ```

2. **Скопируйте URL** из терминала (например: `exp://192.168.1.100:8081`)

3. **Откройте Safari на iPhone**

4. **Введите URL** в адресной строке Safari

5. **Нажмите "Open in Expo Go"** когда появится предложение

## Решение 3: Через настройки Expo Go

1. **Откройте Expo Go**

2. **Перейдите в настройки** (если есть)

3. **Введите URL вручную** или используйте функцию "Enter URL manually"

## Решение 4: Через туннель (если в одной сети)

1. **Убедитесь, что iPhone и Mac в одной Wi-Fi сети**

2. **Запустите Expo с туннелем:**
   ```bash
   npx expo start --tunnel
   ```

3. **Используйте URL из терминала** в Safari на iPhone

## Решение 5: Обновление Expo Go

1. **Откройте App Store**

2. **Найдите "Expo Go"**

3. **Обновите приложение** до последней версии

4. **Перезапустите Expo Go**

## Проверка подключения

После подключения:
- В Expo Go должен появиться проект "mobile" или "DeDato"
- При нажатии на проект должно открыться приложение
- В терминале Expo должны появиться логи подключения

## Если проект в списке, но не запускается

1. **Убедитесь, что Expo dev server запущен:**
   ```bash
   npx expo start
   ```

2. **Проверьте, что iPhone и Mac в одной сети**

3. **Попробуйте удалить проект из списка** и подключиться заново

4. **Проверьте логи в терминале Expo** - там могут быть ошибки подключения

## Альтернатива: Использовать iOS симулятор

Если физическое устройство не работает, можно использовать симулятор:

1. **Установите Xcode** (если еще не установлен)

2. **Запустите симулятор:**
   ```bash
   open -a Simulator
   ```

3. **Запустите Expo:**
   ```bash
   npx expo start
   ```

4. **Нажмите `i`** в терминале Expo для запуска в симуляторе

5. **Maestro будет работать с симулятором** автоматически
