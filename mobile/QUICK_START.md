# 🚀 Быстрый старт для тестирования

## Шаг 1: Проверка .env файла

Убедитесь, что в `mobile/.env` указан правильный API_URL:

```env
API_URL=http://localhost:8000
```

**Для тестирования на реальном устройстве/эмуляторе:**
```env
API_URL=http://192.168.1.100:8000
```
(замените на IP-адрес вашей машины, где запущен backend)

## Шаг 2: Запуск backend (если еще не запущен)

В отдельном терминале:

```bash
cd backend
python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

## Шаг 3: Запуск мобильного приложения

```bash
cd mobile
npm start
```

После этого:
- Откройте **Expo Go** на телефоне
- Отсканируйте QR-код из терминала
- Или нажмите `i` для iOS симулятора / `a` для Android эмулятора

## Шаг 4: Тестовый вход

Используйте тестовые данные:
- **Телефон:** `+79990000001`
- **Пароль:** `test123`

## 🎯 Что проверить в первую очередь

1. ✅ Вход в приложение
2. ✅ Главный экран (для мастера должен быть редирект на `/master`)
3. ✅ Дашборд мастера (`/master`)
4. ✅ Услуги (`/master/services`) - создать/редактировать
5. ✅ Расписание (`/master/schedule`) - переключить слот
6. ✅ Финансы (`/master/finance`) - проверить баланс
7. ✅ Подписки (`/subscriptions`) - проверить оплату

## 🐛 Если что-то не работает

1. Проверьте логи в терминале Metro bundler
2. Проверьте, что backend запущен и доступен
3. Проверьте `.env` файл
4. Перезапустите Metro bundler: `npm start -- --clear`
