# ⚡ Запуск через Expo Go (мгновенно, без сборки)

## Проблема
EAS Build занимает 2+ часа в очереди — слишком медленно для разработки.

## Решение
**Expo Go** — готовое приложение, которое запускает ваш код за секунды.

---

## 🚀 Быстрый старт (2 минуты)

### 1. Установи Expo Go на телефон

- **Android**: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
- **iOS**: [App Store](https://apps.apple.com/app/expo-go/id982107779)

### 2. Запусти Metro bundler

```bash
cd mobile
npm start
```

### 3. Подключи телефон

**Вариант A: QR-код (самый простой)**
1. Открой Expo Go на телефоне
2. Нажми "Scan QR code"
3. Отсканируй QR-код из терминала
4. Приложение загрузится за несколько секунд! 🎉

**Вариант B: По сети**
1. Убедись, что телефон и компьютер в одной Wi-Fi сети
2. В Expo Go: "Enter URL manually"
3. Введи URL из терминала (например: `exp://192.168.0.194:8081`)

**Вариант C: Tunnel (если сети разные)**
```bash
npm start -- --tunnel
```

## 🔄 Hot Reload

После подключения:
- Изменяй код → изменения применяются автоматически
- Перезагрузка: встряхни телефон → меню → "Reload"

---

## ⚠️ Важно

1. **API_URL** — проверь в `mobile/.env`, что указан правильный IP:
   ```
   API_URL=http://192.168.0.194:8001
   ```

2. **Backend должен быть запущен** на указанном адресе

3. **Телефон и компьютер должны быть в одной Wi-Fi сети** (или используй tunnel)

## 📊 Сравнение методов

| Метод | Время | SecureStore | Когда использовать |
|-------|-------|-------------|---------------------|
| **Expo Go** | ⚡ Секунды | ❌ Нет | **Для быстрой разработки** |
| **Локальная сборка** | ⚡⚡ 5-15 мин | ✅ Да | Для тестирования нативных модулей |
| **EAS Build** | ⚡ 2+ часа | ✅ Да | Для финальных сборок |

## 🎯 Рекомендация

**Используй Expo Go для разработки** — это самый быстрый способ.  
**EAS Build используй только для финальных сборок** — когда всё готово к релизу.

## 🐛 Troubleshooting

### "Unable to connect to Metro"
- Проверь, что телефон и компьютер в одной Wi-Fi сети
- Используй `--tunnel`: `npm start -- --tunnel`

### Изменения не применяются
- Встряхни телефон → "Reload"
- Или перезапусти Metro: `npm start`

### "Module not found"
- Некоторые нативные модули не работают в Expo Go
- Используй локальную сборку: `npx expo run:android`
