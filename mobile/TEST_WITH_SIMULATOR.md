## 🚀 Пошаговый запуск

### Шаг 1: Запустить backend

В отдельном терминале:

```bash
cd backend
python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Дождитесь сообщения:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### Шаг 2: Запустить iOS симулятор

```bash
open -a Simulator
```

Или выберите конкретное устройство:
```bash
xcrun simctl boot "iPhone 17 Pro"
```

### Шаг 3: Запустить Expo

В новом терминале:

```bash
cd mobile
npm start
```

Дождитесь появления QR-кода и опций.

### Шаг 4: Открыть приложение в симуляторе

В терминале где запущен `npm start` нажмите:

```
i
```

Приложение автоматически откроется в iOS симуляторе.

## 🔍 Отладка

### Просмотр логов

#### Metro Bundler (терминал)
- Все API запросы: `🌐 [API] Запрос: ...`
- Токен: `🔑 [API] Токен добавлен`
- Ошибки: `❌ [API] Ошибка: ...`

#### iOS Simulator
- **Открыть DevTools:** `Cmd + D` → "Debug Remote JS"
- **Перезагрузить:** `Cmd + R`
- **Открыть меню:** `Cmd + D`

### Частые проблемы

#### Проблема: "Unable to connect to Metro"
**Решение:**
```bash
# Очистить кэш и перезапустить
cd mobile
npm start -- --clear
```

#### Проблема: "Network Error"
**Решение:**
1. Проверьте, что backend запущен: `curl http://localhost:8000/docs`
2. Проверьте `.env` файл: `cat mobile/.env`
3. Перезапустите Metro bundler

#### Проблема: Симулятор не открывается
**Решение:**
```bash
# Закрыть все симуляторы
killall Simulator

# Запустить заново
open -a Simulator
```

#### Проблема: Приложение не загружается
**Решение:**
1. Проверьте, что симулятор запущен:
   ```bash
   xcrun simctl list devices | grep Booted
   ```

2. Перезапустите Expo:
   ```bash
   # Остановить (Ctrl+C)
   # Запустить заново
   npm start
   # Нажать 'i'
   ```

## 🎯 Быстрая команда для запуска

Создайте скрипт для быстрого запуска:

```bash
# Создать файл mobile/start-ios.sh
cat > mobile/start-ios.sh << 'EOF'
#!/bin/bash
echo "🚀 Запуск iOS симулятора..."
open -a Simulator
sleep 2
echo "📱 Запуск Expo..."
cd "$(dirname "$0")"
npm start
EOF

chmod +x mobile/start-ios.sh
```

Использование:
```bash
cd mobile
./start-ios.sh
# Затем нажмите 'i' в терминале
```
