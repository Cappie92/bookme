# 🔍 Как посмотреть логи Network Error

## Проблема: Логи не видны в консоли Metro

Логи могут быть в **DevTools браузера**, а не в консоли Metro bundler.

---

## 📋 Пошаговая инструкция

### 1. Открой DevTools в браузере

**Вариант A: Через приложение**
1. В приложении на телефоне: **встряхни телефон**
2. Выбери **"Debug Remote JS"** или **"Open Debugger"**
3. Откроется Chrome DevTools в браузере

**Вариант B: Через консоль Metro**
1. В консоли Metro bundler нажми: **`j`** (open debugger)
2. Откроется Chrome DevTools

### 2. Перейди на вкладку Console

В открывшемся Chrome DevTools:
1. Перейди на вкладку **"Console"**
2. Там будут все логи, включая:
   - `console.log`
   - `console.error`
   - `console.warn`

### 3. Попробуй войти в приложение

1. Вернись в приложение на телефоне
2. Введи телефон и пароль
3. Нажми "Войти"
4. **Сразу смотри в Console в Chrome DevTools**

**Ты должен увидеть логи:**
```
🔍 [LOGIN] Начало входа
🔍 [LOGIN] Телефон: +7...
🔍 [LOGIN] API_URL: http://192.168.0.194:8001
🌐 [API] Запрос: POST /api/auth/login
❌ [API] NETWORK ERROR: ...
```

---

## 🔧 Альтернатива: Логи в Metro bundler

Если DevTools не открывается, попробуй:

1. **Очисти консоль Metro bundler** (может быть много старых логов)
2. **Перезагрузи приложение** (встряхни телефон → "Reload")
3. **Попробуй войти снова**
4. **Смотри в консоль Metro bundler** — логи должны появиться

---

## 📱 Быстрая проверка

1. **Нажми `j` в консоли Metro bundler** (откроет DevTools)
2. **Перейди на вкладку Console в Chrome DevTools**
3. **Попробуй войти в приложение**
4. **Смотри логи в Console**

---

## 💡 Важно

- **Логи в Metro bundler** — только базовые (запуск, ошибки сборки)
- **Логи в Chrome DevTools Console** — все `console.log`, `console.error` из кода

**Все детальные логи будут в Chrome DevTools Console!**
