# Команды для iOS симулятора

## Где вводить команды

Команды для симулятора можно вводить в **любом терминале** - это не зависит от Expo.

## Вариант 1: В новом терминале

1. **Откройте новый терминал** (не тот, где запущен Expo)
2. **Введите команду:**
   ```bash
   xcrun simctl boot "iPhone 17 Pro"
   ```

3. **Или просто запустите симулятор:**
   ```bash
   open -a Simulator
   ```

## Вариант 2: В том же терминале где Expo

Можно ввести команду в том же терминале, но **в другом окне** или **после остановки Expo** (Ctrl+C).

Но проще открыть **новый терминал**.

## Быстрый способ

Самый простой способ - просто открыть симулятор:

```bash
open -a Simulator
```

Эта команда:
- Откроет симулятор
- Выберет последнее использованное устройство
- Готов к использованию

## После запуска симулятора

1. **В терминале где запущен Expo** нажмите **`i`**
2. **Приложение откроется в симуляторе**

## Проверка что симулятор запущен

```bash
xcrun simctl list devices | grep Booted
```

Должно показать что-то вроде:
```
iPhone 17 Pro (E82D482B-...) (Booted)
```
