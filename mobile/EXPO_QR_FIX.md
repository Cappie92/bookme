# Исправление проблемы с QR-кодом Expo

## Проблема: QR-код не работает при сканировании

Это частая проблема, обычно связанная с сетью или настройками Expo.

## Решения (попробуйте по порядку)

### Решение 1: Использовать туннель (самое надежное)

Остановите текущий Expo (Ctrl+C) и запустите с туннелем:

```bash
cd /Users/s.devyatov/DeDato/mobile
npx expo start --tunnel
```

Туннель работает через интернет, поэтому не требует, чтобы устройства были в одной сети.

**Минус:** Может быть медленнее, но работает всегда.

### Решение 2: Проверить что устройства в одной Wi-Fi сети

1. **На Mac:** Откройте System Settings → Network → Wi-Fi
2. **На iPhone:** Settings → Wi-Fi
3. **Убедитесь что оба подключены к одной сети**

Если сети разные - подключите оба к одной.

### Решение 3: Использовать URL напрямую в Safari

1. **Запустите Expo:**
   ```bash
   npx expo start
   ```

2. **Скопируйте URL** из терминала (например: `exp://192.168.0.194:8081`)

3. **Откройте Safari на iPhone**

4. **Введите URL** в адресной строке Safari

5. **Нажмите "Open in Expo Go"** когда появится предложение

### Решение 4: Проверить Firewall на Mac

Firewall может блокировать подключения:

1. **Откройте System Settings → Network → Firewall**
2. **Временно отключите Firewall** для теста
3. **Или добавьте исключение для Node.js/Expo**

### Решение 5: Использовать LAN режим явно

```bash
cd /Users/s.devyatov/DeDato/mobile
npx expo start --lan
```

Это явно указывает использовать локальную сеть.

### Решение 6: Использовать iOS симулятор (для тестирования)

Если физическое устройство не работает, используйте симулятор:

1. **Откройте симулятор:**
   ```bash
   open -a Simulator
   ```

2. **Запустите Expo:**
   ```bash
   npx expo start
   ```

3. **Нажмите `i`** в терминале Expo

4. **Приложение откроется в симуляторе**

5. **Maestro будет работать с симулятором** автоматически

## Рекомендация для тестирования

Для E2E тестов с Maestro **лучше использовать эмулятор/симулятор**, а не физическое устройство:

### Android эмулятор:
```bash
npx expo start
# Нажмите 'a'
```

### iOS симулятор:
```bash
npx expo start
# Нажмите 'i'
```

Эмуляторы/симуляторы:
- ✅ Всегда доступны для Maestro
- ✅ Не требуют QR-код
- ✅ Работают быстрее для тестирования
- ✅ Не зависят от Wi-Fi сети

## Быстрая проверка

1. **Запустите Expo с туннелем:**
   ```bash
   npx expo start --tunnel
   ```

2. **Отсканируйте QR-код** - должен работать

3. **Или используйте URL** в Safari на iPhone

## Если ничего не помогает

Используйте эмулятор/симулятор для тестирования - это проще и надежнее для Maestro тестов.
