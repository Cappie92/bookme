## ⚠️ Важные замечания

### Фильтрация по времени

**Веб-версия:**
- Показывает все записи в календаре (будущие и прошедшие)
- Фильтрация происходит на уровне UI

**Мобильное приложение:**
- На главном экране фильтрует только будущие записи:
  ```typescript
  const futureBookings = bookings.filter(b => new Date(b.start_time) > now);
  ```
- На экране списка бронирований показывает все записи

**Рекомендация:** Это нормальное поведение — главный экран показывает только ближайшую запись, а полный список доступен в разделе "Бронирования".

## 📌 Примечание

Если запись на 10-12-25 всё ещё не отображается, возможные причины:
1. **Часовой пояс** — проверь, что дата правильно парсится
2. **Статус записи** — убедись, что запись не отменена (`status != 'cancelled'`)
3. **Фильтрация на главном экране** — проверь, что запись действительно в будущем относительно текущего времени
