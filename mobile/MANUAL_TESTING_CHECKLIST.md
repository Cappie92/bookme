# Полноценный чек-лист для ручных тестов мобильного приложения

## Этап 1: Управление избранным

### 1.1. Секция избранного на главном экране

#### Отображение
- [ ] Секция "Избранное" отображается на главном экране
- [ ] Заголовок "Избранное" виден
- [ ] Кнопка раскрытия/сворачивания (+/-) отображается справа от заголовка
- [ ] Кнопка раскрытия/сворачивания работает (меняет состояние)

#### Состояние "Свернуто"
- [ ] При свернутом состоянии список не отображается
- [ ] Если избранных нет, показывается текст "Нет избранных элементов"
- [ ] Кнопка раскрытия/сворачивания все равно видна (если есть избранные)

#### Состояние "Развернуто"
- [ ] При развернутом состоянии отображаются первые 5 элементов
- [ ] Каждый элемент показывает тип (Салон, Мастер, Индивидуальный мастер, Услуга)
- [ ] Каждый элемент показывает название
- [ ] Кнопка удаления (✕) отображается справа от каждого элемента
- [ ] Кнопка "Ещё" отображается, если элементов больше 5
- [ ] Кнопка "Ещё" не отображается, если элементов 5 или меньше

#### Удаление из избранного
- [ ] При нажатии на кнопку удаления (✕) элемент удаляется из списка
- [ ] Список обновляется сразу после удаления
- [ ] Если удален последний элемент, показывается "Нет избранных элементов"
- [ ] Если было 6 элементов и удален один, кнопка "Ещё" исчезает

### 1.2. Модальное окно избранного

#### Открытие
- [ ] Модальное окно открывается при нажатии "Ещё"
- [ ] Заголовок "Моё избранное" отображается
- [ ] Все элементы избранного отображаются в модальном окне

#### Отображение элементов
- [ ] Каждый элемент отображается в карточке
- [ ] Тип элемента отображается с цветным бейджем
- [ ] Название элемента отображается
- [ ] Кнопка удаления (✕) отображается для каждого элемента

#### Пагинация
- [ ] Показывается 10 элементов на странице
- [ ] Если элементов больше 10, отображаются кнопки "Назад" и "Вперед"
- [ ] Отображается номер текущей страницы (например, "1 / 3")
- [ ] Кнопка "Назад" неактивна на первой странице
- [ ] Кнопка "Вперед" неактивна на последней странице
- [ ] При нажатии "Вперед" отображается следующая страница
- [ ] При нажатии "Назад" отображается предыдущая страница
- [ ] Номер страницы обновляется при переходе

#### Удаление в модальном окне
- [ ] При нажатии на кнопку удаления появляется индикатор загрузки
- [ ] После удаления элемент исчезает из списка
- [ ] Если удален последний элемент на странице, происходит переход на предыдущую страницу
- [ ] Если удален не последний элемент, список обновляется на текущей странице
- [ ] После удаления список на главном экране обновляется

#### Закрытие
- [ ] Кнопка "Закрыть" закрывает модальное окно
- [ ] При закрытии модального окна список на главном экране обновляется
- [ ] Можно закрыть модальное окно свайпом вниз (если поддерживается)

### 1.3. Кнопки избранного на страницах

#### На главном экране
- [ ] В секции "Ближайшая запись" кнопка избранного отображается рядом с именем мастера
- [ ] В секции "Ближайшая запись" кнопка избранного отображается рядом с названием салона
- [ ] В секции "Прошедшие записи" кнопка избранного отображается для каждого элемента
- [ ] Кнопка избранного работает (добавляет/удаляет из избранного)
- [ ] Состояние кнопки (заполненное/пустое сердечко) отображается правильно

#### На странице бронирований
- [ ] Кнопка избранного отображается рядом с именем мастера в каждом элементе
- [ ] Кнопка избранного отображается рядом с названием салона в каждом элементе
- [ ] Кнопка избранного работает
- [ ] Состояние кнопки обновляется после добавления/удаления

#### На странице деталей бронирования
- [ ] Кнопка избранного отображается рядом с именем мастера
- [ ] Кнопка избранного отображается рядом с названием салона
- [ ] Кнопка избранного работает
- [ ] Состояние кнопки обновляется

### 1.4. Синхронизация избранного
- [ ] Добавление в избранное на главном экране отражается в модальном окне
- [ ] Добавление в избранное на странице бронирований отражается на главном экране
- [ ] Удаление из избранного в модальном окне отражается на главном экране
- [ ] Состояние избранного сохраняется между сессиями (после перезапуска приложения)

## Этап 2: Детали в отображении бронирований

### 2.1. Элемент бронирования на странице бронирований

#### Основная информация
- [ ] Название услуги отображается крупным шрифтом
- [ ] Имя мастера отображается с подчеркиванием (визуальная ссылка)
- [ ] Название салона отображается (если есть)
- [ ] Название филиала отображается (если есть)

#### Детали бронирования
- [ ] Дата отображается в формате ДД.ММ.ГГГГ
- [ ] Время отображается в формате ЧЧ:ММ - ЧЧ:ММ
- [ ] Длительность услуги отображается (если доступна) в формате "XX мин"
- [ ] Стоимость отображается (если доступна) в формате "XX ₽"
- [ ] Статус оплаты отображается (Оплачено/Не оплачено) с цветовой индикацией
- [ ] Заметки отображаются (если есть)

#### Статус бронирования
- [ ] Статус отображается с цветным бейджем
- [ ] Цвета статусов соответствуют стандартам:
  - Создано - синий
  - Ожидает подтверждения - оранжевый
  - Завершено - зеленый
  - Отменено - красный
  - Ожидает оплаты - желтый

#### Кнопки избранного
- [ ] Кнопка избранного отображается рядом с именем мастера
- [ ] Кнопка избранного отображается рядом с названием салона
- [ ] Кнопки избранного работают корректно

#### Навигация
- [ ] При нажатии на элемент открывается страница деталей бронирования
- [ ] Переход плавный

### 2.2. Страница деталей бронирования

#### Отображение информации
- [ ] ID бронирования отображается в заголовке
- [ ] Статус отображается с цветной индикацией
- [ ] Вся информация о бронировании отображается:
  - Услуга
  - Место (салон/мастер)
  - Филиал (если есть)
  - Адрес (если есть)
  - Мастер (если есть)
  - Дата
  - Время
  - Длительность
  - Стоимость
  - Способ оплаты
  - Статус оплаты
  - Заметки (если есть)
  - Дата создания
  - Дата обновления

#### Кнопки избранного
- [ ] Кнопка избранного отображается рядом с именем мастера
- [ ] Кнопка избранного отображается рядом с названием салона
- [ ] Кнопки избранного работают

#### Визуальные ссылки
- [ ] Имя мастера отображается с подчеркиванием (визуальная ссылка)
- [ ] Визуальная ссылка имеет зеленый цвет (#4CAF50)

## Этап 3: Разделение бронирований на будущие/прошедшие

### 3.1. Страница бронирований

#### Общая структура
- [ ] Страница отображает только будущие бронирования
- [ ] Вкладки "Будущие" и "Прошедшие" НЕ отображаются (убраны)
- [ ] Список можно обновить потягиванием вниз (pull-to-refresh)

#### Элементы бронирования
- [ ] Все элементы отображаются с полной информацией
- [ ] Элементы отсортированы по дате (ближайшие первыми)
- [ ] При нажатии на элемент открывается страница деталей

#### Пустое состояние
- [ ] Если бронирований нет, показывается "Нет записей"
- [ ] Показывается текст "У вас нет будущих записей"
- [ ] Кнопка "Обновить" работает

### 3.2. Секция прошедших записей на главном экране

#### Отображение
- [ ] Секция "Прошедшие записи" отображается всегда
- [ ] Заголовок "Прошедшие записи" виден
- [ ] Показываются последние 3 прошедшие записи
- [ ] Записи отсортированы по дате (новые первыми)

#### Элементы записей
- [ ] Для каждой записи отображается:
  - Название услуги
  - Место (салон/мастер)
  - Дата и время
- [ ] Кнопки избранного отображаются рядом с именами мастеров/салонов
- [ ] Кнопки избранного работают

#### Пустое состояние
- [ ] Если прошедших записей нет, показывается "Нет прошедших записей"
- [ ] Кнопка "Все бронирования" все равно отображается

#### Навигация
- [ ] Кнопка "Все бронирования" ведет на страницу бронирований (будущие)

## Этап 4: Редактирование и изменение времени бронирований

### 4.1. Кнопки редактирования на странице деталей

#### Условия отображения
- [ ] Кнопка "Изменить время" отображается только для будущих бронирований
- [ ] Кнопка "Отменить бронирование" отображается только для будущих бронирований
- [ ] Кнопки НЕ отображаются для прошедших бронирований
- [ ] Кнопки НЕ отображаются для отмененных бронирований
- [ ] Кнопки НЕ отображаются для завершенных бронирований

#### Визуальное оформление
- [ ] Кнопка "Изменить время" имеет зеленый цвет (PrimaryButton)
- [ ] Кнопка "Отменить бронирование" имеет красный цвет
- [ ] Кнопки расположены внизу страницы, после всей информации

### 4.2. Модальное окно изменения времени

#### Открытие
- [ ] Модальное окно открывается при нажатии "Изменить время"
- [ ] Заголовок "Изменить время записи" отображается
- [ ] Модальное окно имеет полупрозрачный фон

#### Календарь
- [ ] Календарь начинается с понедельника
- [ ] Отображается текущая неделя (7 дней)
- [ ] Каждый день показывает:
  - Сокращение дня недели (пн, вт, ср, и т.д.)
  - Число месяца
- [ ] Текущая дата бронирования предвыбрана
- [ ] Выбранная дата подсвечивается зеленым цветом
- [ ] Сегодняшняя дата имеет рамку (если не выбрана)

#### Навигация по неделям
- [ ] Кнопка "‹" (влево) отображается слева
- [ ] Кнопка "›" (вправо) отображается справа
- [ ] Отображается диапазон дат текущей недели (например, "25 дек - 31 дек")
- [ ] При нажатии "›" отображается следующая неделя
- [ ] При нажатии "‹" отображается предыдущая неделя
- [ ] При смене недели выбранная дата обновляется на соответствующий день новой недели

#### Выбор даты
- [ ] При нажатии на день он становится выбранным
- [ ] Выбранная дата подсвечивается зеленым
- [ ] После выбора даты загружаются доступные слоты времени

#### Доступные слоты времени
- [ ] Заголовок "Доступное время [дата]" отображается
- [ ] Показывается индикатор загрузки при загрузке слотов
- [ ] Слоты отображаются в формате "ЧЧ:ММ"
- [ ] Слоты отображаются в виде кнопок
- [ ] Текущее время бронирования НЕ отображается в списке слотов
- [ ] Выбранный слот подсвечивается зеленым
- [ ] Если слотов нет, показывается "Нет доступных слотов на эту дату"

#### Сохранение
- [ ] Кнопка "Сохранить" неактивна, пока не выбран слот времени
- [ ] Кнопка "Сохранить" активна после выбора даты и времени
- [ ] При нажатии "Сохранить" показывается индикатор загрузки
- [ ] После успешного сохранения показывается уведомление "Время бронирования изменено"
- [ ] Модальное окно закрывается
- [ ] Данные бронирования обновляются на странице деталей
- [ ] Время бронирования обновлено

#### Отмена
- [ ] Кнопка "Отмена" закрывает модальное окно
- [ ] При закрытии изменения не сохраняются

### 4.3. Отмена бронирования

#### Подтверждение
- [ ] При нажатии "Отменить бронирование" появляется диалог подтверждения
- [ ] Заголовок диалога: "Отменить бронирование"
- [ ] Текст: "Вы уверены, что хотите отменить это бронирование?"
- [ ] Кнопки: "Нет" (отмена) и "Да, отменить" (подтверждение)

#### Выполнение отмены
- [ ] При нажатии "Нет" диалог закрывается, бронирование не отменяется
- [ ] При нажатии "Да, отменить" показывается индикатор загрузки
- [ ] После успешной отмены показывается уведомление "Бронирование отменено"
- [ ] Происходит редирект на предыдущую страницу (или список бронирований)
- [ ] Бронирование больше не отображается в списке будущих

#### Обработка ошибок
- [ ] При ошибке показывается сообщение об ошибке
- [ ] Бронирование не отменяется при ошибке
