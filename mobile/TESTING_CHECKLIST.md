# Чек-лист для ручной проверки функционала мобильного приложения

## 1. Аутентификация и регистрация

### 1.1. Экран входа/регистрации
- [ ] Отображается слайдер "Вход/Регистрация"
- [ ] Переключение между вкладками "Вход" и "Регистрация" работает корректно
- [ ] Логотип DeDato отображается (изображение, не текст)
- [ ] При открытии клавиатуры форма не перекрывается, можно прокрутить до всех полей
- [ ] Клавиатура скрывается при прокрутке или тапе вне полей

### 1.2. Форма входа
- [ ] Поля "Телефон" и "Пароль" отображаются
- [ ] Вход с корректными данными работает
- [ ] Вход с неверными данными показывает ошибку
- [ ] После успешного входа происходит редирект на главный экран

### 1.3. Форма регистрации
- [ ] Выбор типа аккаунта (Клиент/Мастер) отображается первым
- [ ] Кнопки выбора роли компактные по высоте
- [ ] Поля формы: Тип аккаунта, ФИО, Email, Телефон, Пароль, Подтверждение пароля
- [ ] Валидация email в реальном времени (формат email)
- [ ] Валидация телефона в реальном времени (формат +7XXXXXXXXXX)
- [ ] Валидация пароля в реальном времени (минимум 6 символов)
- [ ] Поле "Подтверждение пароля" проверяет совпадение с паролем
- [ ] Кнопки показа/скрытия пароля работают (иконки глаза, не эмодзи)
- [ ] Чекбокс "Согласие с условиями" обязателен для регистрации
- [ ] Чекбокс "Согласие на обработку персональных данных" опционален
- [ ] Регистрация с корректными данными работает
- [ ] После успешной регистрации происходит редирект на главный экран
- [ ] Новый пользователь создается в базе данных

## 2. Главный экран (Личный кабинет)

### 2.1. Общая информация
- [ ] Приветствие отображается корректно (имя пользователя или "Привет!")
- [ ] Роль пользователя определяется правильно

### 2.2. Секция "Ближайшая запись"
- [ ] Отображается информация о ближайшем бронировании (если есть)
- [ ] Показываются: услуга, место (салон/мастер), дата, время
- [ ] Кнопка избранного (сердечко) отображается рядом с именем мастера/салона
- [ ] Кнопка избранного работает (добавление/удаление из избранного)
- [ ] Кнопка "Все бронирования" ведет на страницу бронирований
- [ ] Если нет будущих бронирований, показывается сообщение "Нет предстоящих бронирований"

### 2.3. Секция "Избранное"
- [ ] Заголовок "Избранное" отображается
- [ ] Кнопка раскрытия/сворачивания (+/-) работает
- [ ] При раскрытии показываются первые 5 элементов избранного
- [ ] Отображаются все типы избранного (салоны, мастера, индивидуальные мастера, услуги)
- [ ] Кнопка "Ещё" появляется, если элементов больше 5
- [ ] Кнопка "Ещё" открывает модальное окно со всеми избранными
- [ ] Кнопка удаления (✕) работает для каждого элемента
- [ ] Если избранных нет, показывается "Нет избранных элементов"

### 2.4. Секция "Прошедшие записи"
- [ ] Заголовок "Прошедшие записи" отображается всегда
- [ ] Показываются последние 3 прошедшие записи
- [ ] Для каждой записи отображаются: услуга, место, дата и время
- [ ] Кнопки избранного (сердечко) отображаются рядом с именами мастеров/салонов
- [ ] Кнопки избранного работают
- [ ] Кнопка "Все бронирования" ведет на страницу бронирований
- [ ] Если прошедших записей нет, показывается "Нет прошедших записей"

### 2.5. Секция "Подписка" (только для мастеров/салонов)
- [ ] Секция отображается только для ролей: master, salon, indie
- [ ] Секция не отображается для роли client
- [ ] Отображается название плана и статус подписки
- [ ] Кнопка "Управление подпиской" ведет на страницу подписок

### 2.6. Секция "Быстрые действия"
- [ ] Кнопка "Мои бронирования" ведет на страницу бронирований
- [ ] Кнопка "Подписка" отображается только для мастеров/салонов
- [ ] Кнопка "Настройки" ведет на страницу настроек

### 2.7. Кнопка выхода
- [ ] Кнопка "Выйти из аккаунта" отображается
- [ ] При нажатии появляется подтверждение
- [ ] После подтверждения происходит выход и редирект на экран логина
