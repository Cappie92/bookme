## 1. Админка - Управление планами подписки

### 1.1 Просмотр планов
- [ ] Войти как администратор
- [ ] Перейти в "Админка" → "Управление подписками" → "Планы подписки"
- [ ] Проверить, что отображаются все планы (Free, Basic, Pro, Premium)
- [ ] Проверить корректность отображения цен, функций и лимитов

### 1.2 Создание плана
- [ ] Нажать "Создать план"
- [ ] Заполнить форму:
  - Название: "Test Plan"
  - Тип: Мастер
  - Цена месяц: 1000₽
  - Цена год: 10000₽
  - Включить функции: Изменение домена, Модули (макс. 2)
- [ ] Сохранить
- [ ] Проверить, что план появился в списке

### 1.3 Редактирование плана
- [ ] Выбрать существующий план
- [ ] Нажать "Редактировать"
- [ ] Изменить цену
- [ ] Сохранить
- [ ] Проверить, что изменения сохранились

### 1.4 Удаление плана
- [ ] Выбрать план
- [ ] Нажать "Удалить"
- [ ] Подтвердить удаление
- [ ] Проверить, что план удален из списка

## 3. Кабинет мастера - Настройки

### 3.1 Информация о подписке
- [ ] Войти как мастер
- [ ] Перейти в "Настройки"
- [ ] Проверить блок с информацией о текущем плане
- [ ] Проверить ссылку "Обновить план"

### 3.2 Изменение домена (Free план)
- [ ] Убедиться, что мастер на плане Free
- [ ] В разделе "Управление сайтом" проверить поле домена
- [ ] Проверить, что поле для изменения домена НЕ доступно
- [ ] Проверить сообщение о необходимости обновить план

### 3.3 Изменение домена (Basic+ план)
- [ ] Обновить план мастера на Basic или выше
- [ ] В разделе "Настройки работы" → "Редактировать настройки"
- [ ] Проверить, что поле "URL домена (изменить)" доступно
- [ ] Попробовать изменить домен на "m-test123"
- [ ] Сохранить
- [ ] Проверить, что домен изменился

### 3.4 Модули на странице (Free план)
- [ ] Убедиться, что мастер на плане Free
- [ ] В разделе "Управление сайтом" проверить блок модулей
- [ ] Проверить, что блокируется добавление модулей
- [ ] Проверить сообщение о необходимости обновить план

### 3.5 Модули на странице (Basic план)
- [ ] Обновить план мастера на Basic (лимит 1 модуль)
- [ ] В разделе "Управление сайтом" проверить блок модулей
- [ ] Нажать "Добавить модуль"
- [ ] Выбрать тип "text"
- [ ] Сохранить
- [ ] Проверить, что модуль добавлен
- [ ] Попробовать добавить второй модуль
- [ ] Проверить, что появляется ошибка о лимите

### 3.6 Модули на странице (Premium план)
- [ ] Обновить план мастера на Premium
- [ ] Добавить несколько модулей
- [ ] Проверить, что лимит не срабатывает (безлимит)
