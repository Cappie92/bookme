# Чеклист: PopupCard hover UX (расписание)

## Изменения (500ms задержка закрытия)

- **MasterScheduleCalendar.jsx**: таймер закрытия 500ms, отмена при наведении на попап
- **PopupCard.jsx**: `onMouseEnter`/`onMouseLeave`, `pointer-events: auto`

## Ручная проверка

1. **Навёл на запись** → попап открылся.
2. **Перевёл курсор на попап** → попап остался открытым.
3. **Увёл курсор и с записи, и с попапа** → закрытие через ~500ms.
4. **Клик "Отменить запись"** → попап не закрывается из-за hover, открывается выбор причины.

## Дополнительно

- При клике внутри попапа (выбор причины, кнопки) hover-логика не мешает.
- Закрытие по крестику работает мгновенно (`handlePopupClose`).
