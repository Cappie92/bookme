**Решение:**
Удалена дублирующая колонка с мастером (строки 1143-1158), оставлена только корректная колонка "Услуга" с `b.service_name`.

**Структура таблицы после исправления:**
```
Header: Мастер | Услуга | Стоимость | Дата | Действия
Row:    (мастер + фаворит) | service_name | price | date | (иконки)
```

**Файл:** `frontend/src/pages/ClientDashboard.jsx`

### 3. Рефактор блока "Мои баллы"

**Было:**
- Карточки с зелёным акцентом
- Крупный текст баллов
- Кнопка "Показать историю" внутри карточки
- Раскрывающаяся история внутри карточки

**Стало:**
- Таблица в стиле "Будущие/Прошедшие записи"
- Header с заголовком "Мои баллы" и кнопкой "Посмотреть все"
- Колонки: Мастер | Баланс | Действия
- Кнопка "История" в колонке "Действия"
- История открывается в модальном окне с таблицей транзакций

**Структура:**
```jsx
<div className="bg-white rounded-lg shadow-sm border p-6">
  <div className="flex justify-between items-center mb-4">
    <h2 className="text-xl font-semibold">Мои баллы</h2>
    <button className="text-green-600 hover:text-green-800 ...">
      Посмотреть все
    </button>
  </div>
  <table>
    <thead>
      <tr>
        <th>Мастер</th>
        <th>Баланс</th>
        <th>Действия</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>{master_name}</td>
        <td>{balance} баллов</td>
        <td>
          <button onClick={handleShowHistory}>История</button>
        </td>
      </tr>
    </tbody>
  </table>
</div>
```

**Модальное окно истории:**
- Заголовок: "История баллов: {master_name}"
- Таблица с колонками: Тип | Услуга | Дата | Баллы
- Типы транзакций: "Начислено" (зелёный) / "Списано" (красный)

**Пустое состояние:**
Изменено с "Баллы начисляются после завершения услуг..." на более короткое "Пока нет начисленных баллов".

**Файлы:**
- `frontend/src/components/ClientLoyaltyPoints.jsx` — полный рефактор
- `frontend/src/pages/ClientDashboard.jsx` — обновлён header секции "Мои баллы"
