# AllBookingsModal — имена не режутся + статус-бейджи

## Изменения

### Задача A — имена не обрезаются без необходимости

**AllBookingsModal.jsx:**
- Убрано `max-w-[140px]` с контейнера имени клиента.
- Контейнер имени: `min-w-0 flex-1 truncate` — занимает оставшееся место, обрезка только при нехватке места.
- Услуга: `min-w-0 truncate` + `maxWidth: 180px` — фиксированный максимум, чтобы оставить место для имени.
- Левая часть строки: `overflow-hidden` для корректного flex-layout.

**MasterDashboardStats.jsx:**
- Блоки «Ближайшие» и «Прошедшие» — аналогично: услуга `maxWidth: 180px`, имя клиента `min-w-0 flex-1 truncate`.

### Задача B — статус для прошедших записей

**Блок B (Требует подтверждения):**
- Бейдж: «Требует подтверждения» (amber-50 / amber-800).

**Блок C (Подтверждённые/отклонённые):**
- Маппинг по `booking.status`:
  - `completed` → «Завершено» (зелёный)
  - `cancelled`, `cancelled_by_client_early`, `cancelled_by_client_late` → «Отменено» (красный)
  - `confirmed` → «Подтверждено» (серый)
  - `created` → «Создано» (серый)
  - `awaiting_confirmation` → «Требует подтверждения» (amber)
  - неизвестный → «Прошло» (серый)

**Стиль бейджа:** `text-xs px-2 py-0.5 rounded-full`, нейтральные цвета.

**Блок A (Будущие):** статус не показывается.
