```javascript
// ДД.ММ.ГГ
function formatDateShort(dateStr) {
  const day = String(d.getDate()).padStart(2, '0')
  const month = String(d.getMonth() + 1).padStart(2, '0')
  const year = String(d.getFullYear()).slice(-2)
  return `${day}.${month}.${year}`
}

// ДД.ММ.ГГ, ЧЧ:ММ
function formatDateTimeShort(dateStr) {
  // ... + hours:minutes
  return `${day}.${month}.${year}, ${hours}:${minutes}`
}
```

### 5. Синхронизация избранного между секциями

**Проблема:**
При клике на сердечко в "Будущих" мастер появлялся в "Избранных", но сердечки в "Прошедших" не обновлялись (и наоборот).

**Решение:**

**A) Добавлен state для Set избранных мастеров:**
```javascript
const [favoriteMasterIds, setFavoriteMasterIds] = useState(new Set())
```

**B) При загрузке данных строится Set:**
```javascript
const masterIds = new Set()
allFavorites.forEach(fav => {
  if (fav.type === 'master' && fav.master_id) {
    masterIds.add(fav.master_id)
  } else if (fav.type === 'indie_master' && fav.indie_master_id) {
    masterIds.add(fav.indie_master_id)
  }
})
setFavoriteMasterIds(masterIds)
```

**C) Обновлён handleFavoriteChange:**
```javascript
const handleFavoriteChange = (type, itemId, isFavorite) => {
  // Обновляем Set для синхронизации
  if (type === 'master' || type === 'indie_master') {
    setFavoriteMasterIds(prev => {
      const newSet = new Set(prev)
      if (isFavorite) {
        newSet.add(itemId)
      } else {
        newSet.delete(itemId)
      }
      return newSet
    })
  }
  // ... остальная логика
}
```

**D) При перезагрузке избранных обновляется Set:**
- В `loadDashboardData()`
- В `handleFavoriteChange()` → `reloadFavorites()`
- В `useEffect()` с зависимостью от `changeCounter`

**Результат:**
- ✅ Toggle favorite в "Будущих" → обновляются "Избранные" и "Прошедшие"
- ✅ Toggle favorite в "Прошедших" → обновляются "Избранные" и "Будущие"
- ✅ Toggle favorite в "Избранных" → обновляются "Будущие" и "Прошедшие"
- ✅ Без перезагрузки страницы
- ✅ Единый источник истины: `favoriteMasterIds` Set
