## Команды для ручной проверки API

```bash
# Проверка статуса скидок
curl -X GET "http://localhost:8000/api/loyalty/status" \
  -H "Authorization: Bearer <TOKEN>"

# Проверка шаблонов
curl -X GET "http://localhost:8000/api/loyalty/templates" \
  -H "Authorization: Bearer <TOKEN>"

# Создание быстрой скидки
curl -X POST "http://localhost:8000/api/loyalty/quick-discounts" \
  -H "Authorization: Bearer <TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{
    "discount_type": "quick",
    "name": "Тестовая скидка",
    "discount_percent": 10,
    "conditions": {"condition_type": "first_visit"},
    "is_active": true,
    "priority": 1
  }'

# Обновление скидки
curl -X PUT "http://localhost:8000/api/loyalty/quick-discounts/{id}" \
  -H "Authorization: Bearer <TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{"discount_percent": 15}'

# Удаление скидки
curl -X DELETE "http://localhost:8000/api/loyalty/quick-discounts/{id}" \
  -H "Authorization: Bearer <TOKEN>"
```
