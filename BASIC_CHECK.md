# Базовая проверка терминала

## Выполните эти команды по одной и проверьте, есть ли вывод:

```bash
# 1. Самая простая команда
echo "test"

# 2. Проверка текущей директории
pwd

# 3. Список файлов
ls

# 4. Проверка, что вы root
whoami

# 5. Проверка даты
date

# 6. Проверка процессов (самая простая)
ps aux | head -5
```

## Если даже эти команды не работают:

1. **Попробуйте другой терминал** - может быть проблема с SSH сессией
2. **Проверьте, что вы действительно на сервере** - выполните `hostname`
3. **Попробуйте выполнить команды через `;` разделитель:**

```bash
echo "test"; pwd; ls; whoami
```

## Альтернативный подход - через веб-интерфейс или другой способ:

Если терминал не работает, возможно:
- Проблема с SSH сессией
- Проблема с правами
- Проблема с оболочкой (shell)

Попробуйте:
1. Переподключиться к серверу
2. Использовать другой SSH клиент
3. Проверить, есть ли веб-интерфейс для управления сервером
