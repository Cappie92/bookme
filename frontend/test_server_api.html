Как работает серверный API

  1. Фронтенд отправляет запрос на `/api/extract-address?url=...`
  2. Сервер анализирует ссылку и определяет её тип
  3. Для обычных ссылок: парсит URL и извлекает адрес
  4. Для коротких ссылок: загружает страницу и парсит HTML
  5. Для координат: использует обратное геокодирование
  6. Возвращает результат в формате JSON
Преимущества серверного API:
  • ✅ Нет ограничений CORS
  • ✅ Поддерживает все типы ссылок
  • ✅ Надежный веб-скрапинг
  • ✅ Обработка ошибок
